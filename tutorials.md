Jag använde den översta (web.dev).

Vad som var bra: Det var bra att det var olika exempel där man kunde se och ändra koden, html och css, och på vissa kunde man trycka på knappar så att man kunde se hur det ändras beroende på vilket värde en property har. Det var mycket förklarande i texten vad olika saker gör. Det var också lite frågor som man kunde svara på för att se om man kunde det.

Vad som var mindre bra: Inte egentligen så dåligt, men det var ganska mycket text. Och det var inga "uppgifter" som man kunde göra.

Vad jag lärt mig:
1. main axis och cross axis
2. direction
3. writing-mode: vertical-rl
4. flex-grow
5. flex-shrink
6. flex-basis
7. nth-child()
8. justify-content
9. align-content
10. align-self
11. align-items

Jag skulle rekommendera den för att det är många exempel och det är lätt att förstå. Man kan också se vad som händer när man ändrar värden på properties.

Betyg: 8/10
